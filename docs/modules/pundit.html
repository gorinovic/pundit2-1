pundit Module

Defined in: src/BaseComponent.js:1

Provides common facilities used by other pundit components, such as callback creation, initialization, logging, etc.

Every component extending this class will be able to use these methods, and the options passed to the constructor.

If the component has an .opts field, it will be used as defaults for the component, overwritable when calling new
